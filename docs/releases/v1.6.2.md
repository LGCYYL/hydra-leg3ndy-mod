## 🆕 Novidades Exclusivas Leg3ndy desta versão

### 🔗 Download Universal por URL *(NOVO)*

A funcionalidade mais pedida: agora você pode colar **qualquer link da internet** diretamente no modal de download de qualquer jogo da biblioteca. O Hydra detecta automaticamente o hoster e baixa com toda a interface nativa (progresso, extração, notificação).

**Hosters suportados automaticamente:**

| URL colada | Detectado como |
|---|---|
| `https://gofile.io/d/...` | Gofile |
| `https://www.mediafire.com/...` | Mediafire |
| `https://pixeldrain.com/u/...` | PixelDrain |
| `https://buzzheavier.com/...` | Buzzheavier |
| `https://fuckingfast.co/...` | FuckingFast |
| `https://qiwi.gg/file/...` | Qiwi.gg *(novo hoster)* |
| `https://datanodes.to/...` | Datanodes |
| `magnet:?xt=...` | Torrent |
| Qualquer `https://` direto | Download Direto |

Como usar: abra qualquer jogo → **Baixar** → role até **🔗 Inserir URL personalizada** → cole o link → clique **Baixar**.

## 🔧 Correções de Estabilidade Leg3ndy

### ✅ Fix: Ícone da Bandeja do Sistema (Tray) Invisível
- O ícone do Hydra na barra de tarefas do Windows estava aparecendo em branco/invisível.
- Corrigido com um ícone branco de alta visibilidade sobre fundo transparente, redimensionado para 16×16 px via força do Windows Manager.

### ✅ Fix: Crash Infinito por Python RPC ausente
- O app entrava em loop de crash-e-restart quando o Python não estava instalado no sistema.
- Adicionado flag `permanentlyFailed`: após a primeira falha, o serviço para de tentar e loga apenas uma vez.

### ✅ Fix: Spam de erros do Native Addon
- O `hydra-native.node` ausente causava erro no console a cada ciclo de polling (2s).
- Adicionado flag `notFound`: se o `.node` não existir, loga uma vez e para de tentar.

---

## 🚀 Destaque Upstream: Addon Nativo em Rust (Hydra Native)

A mudança mais significativa do upstream v3.9.5 é a **remoção do binário externo `aria2c`** e sua substituição por um **addon nativo compilado em Rust** (`hydra-native`), carregado diretamente pelo processo Node.js do Electron.

- ✅ **Sem `aria2c.exe` externo** — menos processos externos, menos superfície de ataque
- ✅ **Menor tamanho do instalador** — aria2c era um binário separado
- ✅ **Download integrado ao processo** — comunicação via FFI, sem overhead de IPC
- ✅ **Sem processos órfãos** ao fechar o app
