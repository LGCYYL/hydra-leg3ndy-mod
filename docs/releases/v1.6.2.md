## 🚀 Destaque: Addon Nativo em Rust (Hydra Native)

A mudança mais significativa desta versão é a **remoção do binário externo `aria2c`** e sua substituição por um **addon nativo compilado em Rust** (`hydra-native`), carregado diretamente pelo processo Node.js do Electron.

**O que isso significa na prática:**
- ✅ **Sem `aria2c.exe` externo** — menos processos externos, menos superfície de ataque
- ✅ **Menor tamanho do instalador** — aria2c era um binário separado distribuído com o app
- ✅ **Download integrado ao processo** — comunicação via FFI, sem overhead de IPC entre processos
- ✅ **Gerenciamento de ciclo de vida simplificado** — sem processos órfãos ao fechar o app

## 🔧 Novidades e Correções da v1.6.2

### ✅ Refactor: Migração Completa para Addon Nativo (Rust)
- **Antes:** O gerenciador de downloads usava `aria2c` como processo filho, comunicando via JSON-RPC.
- **Agora:** Um novo módulo Rust (`native/hydra-native`) é compilado como `.node` addon e carregado nativamente. O arquivo `src/main/services/aria2.ts` foi removido e substituído pelo `native-addon.ts`.

### ✅ Fix: Gestão de Processos Python RPC
- O serviço `python-rpc.ts` foi atualizado para melhor ciclo de vida e robustez ao inicializar o servidor Python para torrenting.

### ✅ Fix: Lógica de Progresso de Download Aprimorada
- A detecção de download completo agora verifica corretamente flags `isCheckingFiles` e `isDownloadingMetadata` além do progresso `=== 1`, evitando falsos gatilhos de "conclusão" durante verificação de arquivos e metadados.

### ✅ Fix: Downloads com Status "Removed" Ignorados
- A página de Downloads agora filtra corretamente entradas com status `removed`, evitando que apareçam na lista.

### ✅ Fix: Setup.py com Includes de OpenSSL no Windows
- A build do Python RPC agora inclui automaticamente as DLLs do OpenSSL (`libcrypto`, `libssl`) ao compilar no Windows, corrigindo erros de runtime em ambientes sem Python no PATH do sistema.

### ✅ Novo: Atalhos Steam para Jogos
- Interface de criação/remoção de atalhos Steam agora disponível, com suporte a flag VR.

### ✅ Novo: Locale Japonês (ja)
- Adicionado suporte completo ao idioma Japonês (`src/locales/ja/translation.json`).
